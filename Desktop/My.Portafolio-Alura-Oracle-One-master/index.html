Proyectos

Encriptado de Texto

Este proyecto consiste en una herramienta web que permite encriptar y desencriptar texto utilizando un algoritmo simple de sustitución de caracteres. El usuario puede ingresar un texto, elegir entre encriptarlo o desencriptarlo con solo un clic, y copiar el resultado al portapapeles. La aplicación está diseñada para aceptar solo letras minúsculas y sin acentos, y proporciona un método práctico para experimentar con la encriptación de datos de manera interactiva.

AluraPhone - E-commerce

Este proyecto es una tienda en línea para la venta de productos tecnológicos como celulares, smartwatches y accesorios. Los usuarios pueden explorar productos, ver promociones y realizar búsquedas filtradas, todo con un diseño responsivo. Además, cuenta con una sección de contacto para la comunicación con la empresa. El sistema utiliza una base de datos .json que permite a los usuarios registrarse y gestionar productos, incluyendo la posibilidad de agregar, editar o eliminar artículos. Este proyecto demuestra habilidades en diseño de interfaces y gestión dinámica de contenido.

Gestión de Equipos - React.js

Este proyecto es una aplicación de gestión de equipos de trabajo en React.js, que permite crear, editar y eliminar colaboradores dentro de distintos equipos. Los usuarios pueden registrar nuevos colaboradores, asignarlos a equipos específicos y cambiar los colores asociados a cada equipo. Además, pueden marcar a sus colaboradores favoritos y eliminar aquellos que ya no forman parte del equipo. La aplicación hace uso de useState para gestionar el estado de los colaboradores y equipos, mientras que se utilizan componentes como Formulario, MiOrg, Equipo, y Header para estructurar la interfaz. Cada colaborador es identificado de manera única por un UUID, y la lógica de agregar, editar o eliminar se maneja de manera dinámica. Este proyecto te permite practicar el uso de hooks en React, como useState, y trabajar con la manipulación de listas y eventos en la interfaz.
